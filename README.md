# Awesome AI Scientist

A verified map of AI Scientist systems, papers, benchmarks, and open-source implementations.

This repository tracks the fast-moving AI Scientist / Auto Research ecosystem. It focuses on resources that are useful for building, evaluating, or understanding research agents.

## Scope

- AI Scientist and Auto Research systems
- Scientific discovery agents
- Research automation papers
- Open-source implementations
- AI4S and research-agent benchmarks
- Survey papers and technical reports

## Categories

- **Systems**: end-to-end AI Scientist, co-scientist, and auto-research agents.
- **Components**: search planners, paper rankers, evidence extractors, hypothesis generators, critics, and experiment agents.
- **Benchmarks**: AI4S, scientific discovery, ML research, paper reproduction, data analysis, and long-horizon agent benchmarks.
- **Models**: models trained or adapted for scientific reasoning, literature search, code research, or research-agent workflows.
- **Surveys**: review papers and ecosystem maps.

## Verification Standard

Each important entry should make the verification state explicit:

- source link
- paper link, if available
- code link, if available
- open-source status
- benchmark or evaluation used
- reproducibility notes
- last verified date

## Contributing

Pull requests are welcome. Prefer entries with primary sources, runnable code, benchmark results, or clear evidence of real usage.

Avoid unsourced claims, marketing-only announcements, and duplicate list entries without additional verification value.
